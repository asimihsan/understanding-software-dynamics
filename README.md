# My C Project

This is a sample C project using CMake for building and maintaining code snippets.

## Directory Structure

```
my_c_project/
├── CMakeLists.txt
├── build/
├── src/
│   ├── main.c
│   └── snippet1.c
├── include/
│   └── snippet1.h
└── README.md
```

## Building the Project

```sh
mkdir -p build
cd build
cmake ..
make
```

## Running the Project

```sh
./my_c_project
```
